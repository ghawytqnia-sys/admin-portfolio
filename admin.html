How this works

No server, no database — your edits are written to localStorage in this browser and read by index.html on load.

For deployment (e.g. GitHub Pages / Netlify), edit content here, then use Backup & Restore → Export to download a JSON snapshot, and hand that file to a developer to bake into js/data.js / js/translations.js permanently — or keep using this panel per-browser for quick local edits.
